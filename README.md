## Tightknit mock api

This is a mock api for the Tightknit External API, with a Fiberplane playground to play with the (mock) api.

Visit `/fp` to explore API routes and make requests.

## Running the app

```bash
pnpm dev
```

## Deploying the app

```bash
pnpm run deploy
```
